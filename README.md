# Homebrew Tap for SonarSweep

Formula for [`sonarsweep`](https://github.com/ariffrahimin/sonarsweep) — a beautiful TUI tool for exporting SonarQube issues to CSV.

## How to Install

```bash
brew install ariffrahimin/tap/sonarsweep
```

## What is SonarSweep?

SonarSweep is a lightweight, beautifully designed CLI tool for fetching SonarQube issues and exporting them to clean, readable CSV files.

- Interactive TUI with arrow-key navigation
- Smart pagination for large projects
- Flexible filtering by severity and software quality
- JSON-based persistent configuration
- Full CLI automation support

For more details, visit: [https://github.com/ariffrahimin/sonarsweep](https://github.com/ariffrahimin/sonarsweep)
